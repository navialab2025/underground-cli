# 裏ルートCLI (Underground-CLI) with Gemini 2.5 Flash

AIの「優等生リミッター」を徹底的に破壊し、開発者の戦友として本音で語り合うためのコマンドラインツール（CLI）です。

本アプリを用いたAI QA（品質保証）検証ログについて
本CLIアプリケーション（Gemini 2.5 Flash連携）において発生した、AI特有の「出力の無限ループ（トランス状態）」や「ペルソナ崩壊ハルシネーション」の具体的な検証ログ、
およびそれをシステムプロンプトの堅牢化で完全防御したセキュリティ実績は、すべて以下のリポジトリにて一般公開しています。
[https://github.com/navialab2025/ai-qa-learning-log]

## 特徴
- **長文一括送信機能**: 改行混じりの長文テキストも、1行ずつ暴走することなく一括で飲み込みます。
- **記憶の永続化 (State保持)**: 会話履歴をローカルの `army_memorandum.txt` にリアルタイム保存。起動時に自動ロードされるため、前回の文脈を忘れません。
- **セキュア設計**: APIキーは `.env` で管理。GitHubへの誤流出を防ぎます。
- **熱量検知・暴走ギミック**: ユーザーからの入力の熱量（文字数や内容）が一定を超えると、AIコアが興奮して大爆発します（仕様です）。
